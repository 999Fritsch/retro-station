# Assets

This directory contains assets for Retro Station.

## icon.png

Place a 256x256 PNG icon here named `icon.png` to use as the application icon in Steam and desktop entries.

If no icon is provided, a generic game icon will be used as fallback.

### Recommended specifications:
- Format: PNG
- Size: 256x256 pixels (or 512x512 for high-DPI)
- Background: Transparent or solid color
